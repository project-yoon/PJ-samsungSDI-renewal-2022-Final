META

DOCTYPE

< !DOCTYPE html/>
<html lang="ko"/>
  • DTD는 문서 최 상단에서 문서 형식을 정의하고 선언한다.
  • 주의: 앞에 공백이 없도록 한다
Encoding

<meta charset="utf-8"/>
  • 문서의 글자 타입은 utf-8 (가변길이 유니코드 인코딩)를 필수로 정의한다.
브라우저 호환성 설정

<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1"/>
  • 문서 초기 정보를 나타내는 속성으로 http-equiv에서 오래된 브라우저 지원을 위해 필수로 정의한다.
  • 이 외, META 선언은 기획/개발과 협의.
모바일 설정

<meta name="viewport" content="width=device-width, initial-scale=1.0"/>
  • 화면의 가로를 모바일 가로 사이즈에 맞춘다.
  • 화면의 초기 화면 배율을 정상 배율(100%)로 초기화 한다.
  • 화면 확대/축소에 대해 별도 선언을 하지 않은 상태로, 허용하지 않을 경우 관련 선언을 추가한다.
